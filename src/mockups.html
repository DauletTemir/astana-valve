Neo-Brutalism

Толстые границы, контрастные блоки, жирная типографика. Фильтруется в мейнстрим из нишевого движения. Идеально для B2B — проецирует уверенность и директность. Применено в Макете 1.

Bento Grid

Модульная сетка-карточки вместо классических секций. Dark Mode как первоклассная тема. Bento + темный фон = доминирующий формат 2025–2026. Применено в Макете 2.

Kinetic Typography

Шрифт как главный архитектурный элемент вместо hero-фото. Viewport-scaled fonts, oversized headings, анимация на скролле. Меньше изображений = быстрее загрузка. Применено в Макете 3.

Mobile-First + Мультиязычность

WCAG 2.2 AA — теперь юридическое требование. Мгновенный языковой переключатель (RU/KZ/EN) без перезагрузки страницы. Семантический HTML + Schema.org для AI-поисковиков.
